# tinyfile

Tiny file is a tiny file compressor service leveraging web technologies to compress files without the files ever leaving
your browser.

<img src="public/screenshot.png" alt="MarineGEO circle logo" style="width: 720px;"/>

# How to use

```shell
$ npm install
$ npm run dev
```

# How to build

```shell
$ npm run build
```
